# Syndra MySQL Client

基于 PyQt6 开发的轻量级 MySQL 客户端工具，提供直观的图形界面管理 MySQL 数据库。

## 功能特性

✅ **连接管理**
- 保存多个数据库连接配置
- 密码加密存储
- 连接测试功能
- SSH 隧道支持（预留接口）

✅ **数据库浏览**
- 树形视图展示数据库和表
- 双击打开表数据浏览
- 支持在新标签页打开多个表
- 分页浏览（支持 10-100 条/页自定义）
- 表操作：重命名、删除、查看结构

✅ **SQL 编辑器**
- SQL 语法高亮
- 自动补全（关键字 + 函数 + **表名自动补全**）
- **SQL 格式化美化**（需要 `sqlparse` 库）
- **支持多个 SQL 查询标签页**
- 查询结果表格显示

✅ **数据操作**
- 表格列宽度**支持手动调整**
- 右键**复制功能**：复制单元格 / 复制整行 / 复制全部
- 结果可以直接复制粘贴到 Excel/Markdown

## 使用说明

### 连接数据库
1. 点击左侧 **新建连接**
2. 填写连接信息：连接名、主机、端口、用户名、密码
3. 可以点击 **测试连接** 验证连接
4. 点击 **连接** 连接数据库，可选择保存连接配置

### 浏览数据
1. 连接成功后会加载所有数据库和表
2. **双击** 表名在新标签页打开数据浏览
3. 支持分页浏览，可以调整每页显示条数
4. 表格右键表头分隔线可以**手动调整列宽**
5. 右键单元格可以**复制**单元格/整行/全部数据

### 执行 SQL 查询
1. 默认有一个 SQL Editor 标签页
2. 在菜单栏选择 **查询 → 新建SQL查询** 可以打开多个查询标签
3. 输入 SQL 后点击 **执行SQL**
4. SELECT 查询结果会显示在下方表格
5. 点击 **格式化SQL** 美化 SQL（需要 `sqlparse`）

### 代码重构
本项目经过完整重构：
- 原单体文件拆分为模块化结构
- 消除代码重复（表格浏览对话框和标签页共享逻辑）
- 所有代码添加完整注释
- 提高可维护性和扩展性

### 安全
- 保存连接时密码使用 Fernet 对称加密存储
- 密钥基于固定密码导出，满足基本安全需求
